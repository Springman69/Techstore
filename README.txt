TechStore – ASP.NET e-commerce prototype

Projekt akademicki realizowany zespołowo (4 osoby) w ramach studiów informatycznych.  
Celem było stworzenie prototypu sklepu internetowego w technologii ASP.NET z wykorzystaniem SQLite jako bazy danych.

Rekomendowane jest skopiowanie zawartości folderu "Baza_danych" do lokalizacji "C:\" 

Przykład użycia:
C:\TechStoreDatabase\Database.db
